Tipos de Datos
Concepto Descripción Ejemplo
Tipos de Datos En Python, los tipos de datos básicos incluyen enteros, flotantes, cadenas de texto, booleanos, etc. Python es dinámicamente tipado, por lo que no necesitas declarar el tipo. 
a = 5  # Entero (int)
b = 3.14  # Flotante (float)
c = "Hola"  # Cadena (str)
d = True  # Booleano (bool)
print(type(a))  # Output: <class 'int'>
Estructuras de Datos
Concepto Descripción Ejemplo
Estructuras de Datos Python ofrece estructuras como listas, tuplas, diccionarios y conjuntos para almacenar colecciones de datos. 
lista = [1, 2, 3]  # Lista (mutable)
tupla = (1, 2, 3)  # Tupla (inmutable)
diccionario = {"clave": "valor"}  # Diccionario
conjunto = {1, 2, 3}  # Set (únicos, no ordenados)
print(lista[0])  # Output: 1
Operadores Aritméticos
Concepto Descripción Ejemplo
Operadores Aritméticos Los operadores aritméticos permiten realizar cálculos matemáticos: suma (+), resta (-), multiplicación (*), división (/), módulo (%), exponente (**), división entera (//). 
a = 10
b = 3
print(a + b)  # Suma: 13
print(a - b)  # Resta: 7
print(a * b)  # Multiplicación: 30
print(a / b)  # División: 3.333...
print(a % b)  # Módulo: 1
print(a ** b)  # Exponente: 1000
print(a // b)  # División entera: 3
Operadores Lógicos
Concepto Descripción Ejemplo
Operadores Lógicos Los operadores lógicos se usan para combinar expresiones booleanas: and (y), or (o), not (no). 
a = True
b = False
print(a and b)  # False
print(a or b)  # True
print(not a)  # False
Condicionales
Concepto Descripción Ejemplo
Condicionales Las condicionales permiten ejecutar código basado en condiciones: if, elif, else. 
edad = 18
if edad >= 18:
    print("Mayor de edad")
elif edad >= 13:
    print("Adolescente")
else:
    print("Menor")  # Output: Mayor de edad
Estructuras de Control
Concepto Descripción Ejemplo
Estructuras de Control Incluyen bucles como for y while para repetir acciones, y break/continue para controlar el flujo. 
# For loop
for i in range(3):
    print(i)  # 0 1 2

# While loop
cont = 0
while cont < 3:
    print(cont)
    cont += 1  # 0 1 2

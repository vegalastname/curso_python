Ejercicio 1: Sumar dos números
Crea un algoritmo que tome dos números como entrada y devuelva su suma.
Pseudocódigo:
Leer número1
Leer número2
suma = número1 + número2
Imprimir suma
Implementación en Python:
numero1 = int(input("Ingrese el primer número: "))
numero2 = int(input("Ingrese el segundo número: "))
suma = numero1 + numero2
print("La suma es:", suma)
Explicación: Usamos 'input()' para leer entradas del usuario, convertimos a int con 'int()'. Sumamos con '+'. Imprimimos con 'print()'.
Ejercicio 2: Encontrar el máximo de dos números
Crea un algoritmo que compare dos números y devuelva el mayor.
Pseudocódigo:
Leer número1
Leer número2
Si número1 > número2 entonces
    máximo = número1
Sino
    máximo = número2
Imprimir máximo
Implementación en Python:
numero1 = int(input("Ingrese el primer número: "))
numero2 = int(input("Ingrese el segundo número: "))
if numero1 > numero2:
    print("El máximo es:", numero1)
else:
    print("El máximo es:", numero2)
Explicación: 'if' compara con '>', 'else' maneja el otro caso. 'print()' muestra el resultado.
Ejercicio 3: Verificar si un número es par o impar
Crea un algoritmo que determine si un número es par o impar.
Pseudocódigo:
Leer número
Si número modulo 2 == 0 entonces
    Imprimir "Par"
Sino
    Imprimir "Impar"

Ejercicio 4: Calcular factorial
Crea un algoritmo que calcule el factorial de un número.
Pseudocódigo:
Leer número
factorial = 1
Para i desde 1 hasta número
    factorial = factorial * i
Imprimir factorial
Implementación en Python:
numero = int(input("Ingrese un número: "))
factorial = 1
for i in range(1, numero + 1):
    factorial *= i
print("El factorial es:", factorial)
Explicación: 'for' con 'range()' itera. '*=' multiplica y asigna.
Ejercicio 5: Secuencia de Fibonacci
Crea un algoritmo que genere la secuencia de Fibonacci hasta un número dado.
Pseudocódigo:
Leer n
a = 0
b = 1
Imprimir a
Imprimir b
Para i desde 2 hasta n
    siguiente = a + b
    Imprimir siguiente
    a = b
    b = siguiente
Implementación en Python:
n = int(input("Ingrese el número de términos: "))
a, b = 0, 1
print(a)
print(b)
for i in range(2, n):
    siguiente = a + b
    print(siguiente)
    a = b
    b = siguiente
Explicación: Asignación múltiple 'a, b = 0, 1'. 'for' itera, suma y reasigna.

Ejercicio 7: Contar vocales en una cadena
Crea un algoritmo que cuente el número de vocales en una cadena.
Pseudocódigo:
Leer cadena
contador = 0
vocales = "aeiouAEIOU"
Para cada caracter en cadena
    Si caracter está en vocales
        contador = contador + 1
Imprimir contador
Implementación en Python:
cadena = input("Ingrese una cadena: ")
vocales = "aeiouAEIOU"

    Imprimir "Es palíndromo"

Ejercicio 11: Ordenar una lista (burbuja)
Crea un algoritmo que ordene una lista usando el método de burbuja.
Pseudocódigo:
lista = [3, 1, 4, 2]
n = longitud de lista
Para i desde 0 hasta n-1
    Para j desde 0 hasta n-i-2
        Si lista[j] > lista[j+1]
            Intercambiar lista[j] y lista[j+1]
Imprimir lista
Implementación en Python:
lista = [3, 1, 4, 2]
n = len(lista)
for i in range(n):
    for j in range(0, n-i-1):
        if lista[j] > lista[j+1]:
            lista[j], lista[j+1] = lista[j+1], lista[j]
print("Lista ordenada:", lista)
Explicación: Dos 'for' anidados comparan e intercambian adyacentes. 'len()' obtiene longitud.
Ejercicio 12: Buscar un elemento en una lista
Crea un algoritmo que busque un elemento en una lista y devuelva su posición.
Pseudocódigo:
lista = [1, 2, 3, 4]
elemento = 3
posicion = -1
Para i desde 0 hasta longitud de lista - 1
    Si lista[i] == elemento
        posicion = i
        Salir
Si posicion != -1
    Imprimir "Encontrado en posición" posicion
Sino
    Imprimir "No encontrado"
Implementación en Python:
lista = [1, 2, 3, 4]
elemento = 3
posicion = -1
for i in range(len(lista)):
    if lista[i] == elemento:
        posicion = i
        break
if posicion != -1:
    print("Encontrado en posición:", posicion)
else:
    print("No encontrado")
Explicación: 'range(len(lista))' itera índices. 'break' sale al encontrar.

Leer número
contador = 0
Mientras número > 0
    número = número / 10 (entero)

Ejercicio 17: Verificar si un número es primo
Crea un algoritmo que verifique si un número es primo.
Pseudocódigo:
Leer número
Si número <= 1
    No es primo
Sino
    es_primo = verdadero
    Para i desde 2 hasta raíz cuadrada de número
        Si número modulo i == 0
            es_primo = falso
            Salir
    Si es_primo
        Imprimir "Es primo"
    Sino
        Imprimir "No es primo"
Implementación en Python:
import math
numero = int(input("Ingrese un número: "))
if numero <= 1:
    print("No es primo")
else:
    es_primo = True
    for i in range(2, int(math.sqrt(numero)) + 1):
        if numero % i == 0:
            es_primo = False
            break
    if es_primo:
        print("Es primo")
    else:
        print("No es primo")
Explicación: 'import math' para 'sqrt()'. 'for' verifica divisores hasta raíz cuadrada. 'break' optimiza.
Ejercicio 18: Calcular MCD de dos números
Crea un algoritmo que calcule el Máximo Común Divisor (MCD) de dos números.
Pseudocódigo:
Leer a, b
Mientras b != 0
    temp = b
    b = a modulo b
    a = temp
Imprimir a
Implementación en Python:
a = int(input("Ingrese el primer número: "))
b = int(input("Ingrese el segundo número: "))
while b != 0:
    temp = b
    b = a % b
    a = temp
print("MCD:", a)
Explicación: Algoritmo de Euclides con 'while' y '%' para módulo.
Ejercicio 19: Calcular potencia de un número
Crea un algoritmo que calcule la potencia de un número (base^exponente).
Pseudocódigo:
Leer base, exponente
resultado = 1
Para i desde 1 hasta exponente
    resultado = resultado * base
Imprimir resultado

base = int(input("Ingrese la base: "))

Ejercicio 20: Imprimir patrón de estrellas (triángulo)
Crea un algoritmo que imprima un triángulo de estrellas dado un número de filas.
Pseudocódigo:
Leer filas
Para i desde 1 hasta filas
    Para j desde 1 hasta i
        Imprimir "*"
    Imprimir línea nueva
Implementación en Python:
filas = int(input("Ingrese el número de filas: "))
for i in range(1, filas + 1):
    for j in range(i):
        print("*", end="")
    print()
Explicación: 'for' anidado imprime '*'. 'end=""' evita saltos, 'print()' nueva línea.

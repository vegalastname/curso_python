Qué es una Función
Una función en Python es un bloque de código reutilizable que realiza una tarea específica. Se define con 'def' y puede recibir parámetros y retornar valores.
Ejemplo:
def suma(a, b):
    return a + b

resultado = suma(5, 3)
print(resultado)  # Output: 8
Qué es un Algoritmo
Un algoritmo es una secuencia de pasos lógicos para resolver un problema. En programación, es el "plan" que sigue el código para lograr un objetivo, como ordenar una lista.
Ejemplo (Algoritmo de burbuja para ordenar):
def burbuja(lista):
    n = len(lista)
    for i in range(n):
        for j in range(0, n-i-1):
            if lista[j] > lista[j+1]:
                lista[j], lista[j+1] = lista[j+1], lista[j]
    return lista

print(burbuja([3, 1, 2]))  # Output: [1, 2, 3]
Qué es el Pseudocódigo
El pseudocódigo es una descripción de alto nivel de un algoritmo o programa, escrita en un lenguaje similar al humano pero con estructuras de programación. No es código ejecutable, sino una forma de planificar la lógica antes de implementarla en un lenguaje real como Python. Ayuda a enfocarse en la lógica sin preocuparse por la sintaxis específica.
Características:
  • Usa palabras clave como "si", "entonces", "para", "mientras".
  • No depende de un lenguaje de programación específico.
  • Es útil para diseñar algoritmos y depurar lógica.
Ejercicio FizzBuzz
FizzBuzz es un ejercicio clásico para practicar lógica de programación. El objetivo es imprimir números del 1 al 100, pero reemplazar múltiplos de 3 por "Fizz", múltiplos de 5 por "Buzz", y múltiplos de ambos por "FizzBuzz".
Pseudocódigo:
Para cada número i desde 1 hasta 100:
    Si i es múltiplo de 3 y de 5:
        Imprimir "FizzBuzz"
    Sino si i es múltiplo de 3:
        Imprimir "Fizz"
    Sino si i es múltiplo de 5:
        Imprimir "Buzz"
    Sino:
        Imprimir i
Implementación en Python:
for i in range(1, 101):
    if i % 3 == 0 and i % 5 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
Este código usa un bucle for, condicionales y operadores módulo (%). Ejecutándolo, obtendrías una lista como: 1, 2, Fizz, 4, Buzz, ..., FizzBuzz para 15, etc.
